roverpi

python software for remotely controlling a raspberry pi powered rover

  • Home
  • Downloads
  • Doc
  • Hardware

roverpi is a set of python scripts that allows one to send commands over the network to a rapsberry pi-powered drone using a GUI Client, like you would control a RC car using the keyboard. The client is also able to display a video stream from a camera set on the rover. Planned features include sending text-to-speech commands and playing adio files from the rover. It was originally developped for a 4WD chassis with tracks (a "tank") but could probably be adapted to oher raspberry pi-GPIO based projects. Code for both the rover and the client are freely available for anyone to use and/or modify.
